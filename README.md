# Solidity Code Co-Pilots

Use AI to review your solidity code.

![Solidity Code Co-Pilots](./public/Solidity-Co-Pilots.png)

## Running Locally

**1. Clone Repo**

```bash
git clone https://github.com/thopatevijay/solidity-co-pilots.git
```

**2. Install Dependencies**

```bash
npm i
```

**3. Run App**

```bash
npm run dev
```
